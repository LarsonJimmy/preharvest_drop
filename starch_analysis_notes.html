Introduction

This is a file that describes the process for analyzing fruit firmness for a preharvest drop study conducted from Aug.-Oct.2020 on Red Delicious apples. 12 total fruit from the outside 2 trees per plot these fruit had their internal ethylene concentrations and firmness measured, juice from each apple in a treatment by rep combination was collected and had their brix measured using a refractometer. Following firmness, each fruit was cut in half equatorially and dipped into a starch iodine solution. Staining pattern was evaluated on a 1-8 scale:

  • August 28

  • September 1

  • September 7

  • September 14

  • September 21

  • September 29

  • October 12

Join Data

Data was joined for each measurement date starting with the first, Aug.28

starchAug28 %>%
  rename(starchAug28 = starch) %>%
  left_join(starchSep1) %>%
  rename(starchSep1 = starch) %>%
  left_join(starchSep7) %>%
  rename(starchSep7 = stach) %>%
  left_join(starchSep14) %>%
  rename(starchSep14 = starch) %>%
  left_join(starchSep21) %>%
  rename(starchSep21 = starch) %>%
  left_join(starchSep29) %>%
  rename(starchSep29 = starch) %>%
  select(treatment, rep, fruit, starchAug28, starchSep1, starchSep7, starchSep14, starchSep21, starchSep29) %>%
  left_join(starchOct12) %>%
  rename(starchOct12 = starch) %>%
  select(treatment, rep, fruit, starchAug28, starchSep1, starchSep7, starchSep14, starchSep21, starchSep29,
         starchOct12) -> starch

ANOVA Assumptions

ANOVA Model

starchAug28.aov <- aov(starchAug28 ~ treatment, data = starch)

Normality Check

starchAug28.resid <- residuals(object = starchAug28.aov)
shapiro.test(x = starchAug28.resid)

Homogeneity of Variance Check

leveneTest(starchAug28 ~ treatment, data = starch)
